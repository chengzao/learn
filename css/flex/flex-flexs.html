flex

flex: 属性是flex-grow(放大,0), flex-shrink(缩小,1) 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。

flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

flex: 该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)
